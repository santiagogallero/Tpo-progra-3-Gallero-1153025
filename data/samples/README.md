# Ejemplos de entrada para los algoritmos

Este directorio contiene archivos de ejemplo para probar los algoritmos.

## Estructura

- `input1.txt` - Ejemplo simple
- `input2.txt` - Ejemplo con más datos
- `input3.txt` - Caso límite

## Nota

Los ejemplos están integrados en el código de `main.py`, por lo que no es necesario usar archivos de entrada para las demostraciones.

Sin embargo, estos archivos pueden ser útiles para:
- Testing adicional
- Benchmarking
- Casos de uso personalizados
